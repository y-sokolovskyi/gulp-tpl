#### Installation

- > nvm use 14+
- > npm i

#### Development

- > npm run dev

#### Start server on the 4200 port

- > gulp/tasks/server.js -> port: 4200
- > http://localhost:4200/ss

#### Create zip

- > npm run zip

#### Build

- > npm run build

#### Deploy to FTP

- > Add config to gulp/config/ftp.js
- > npm run deploy
